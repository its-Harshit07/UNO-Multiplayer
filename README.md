# 🎮 UNO Multiplayer

A modern real-time multiplayer UNO game built with **React, TypeScript, Socket.IO, Express, Prisma, and PostgreSQL**.

Play with friends using room codes, invite links, AI bots, and spectator mode.

## ✨ Features

- 🎲 Real-time Multiplayer
- 🤖 AI Bot Players
- 👥 2–10 Players
- 🔗 Invite Links
- 👀 Spectator Mode
- 📱 Mobile Responsive
- 🎵 Sound Effects
- ⚡ Fast Socket.IO Gameplay
- 🃏 Complete UNO Rules

## 🛠️ Tech Stack

**Frontend**
- React
- TypeScript
- Vite
- Framer Motion

**Backend**
- Express
- Socket.IO
- Prisma ORM
- PostgreSQL

**Deployment**
- Vercel
- Render
- Neon PostgreSQL
